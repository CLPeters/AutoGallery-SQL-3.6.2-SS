AutoGallery SQL Documentation
General Information

This documentation covers all aspects of AutoGallery SQL's operation, and tries to be as thorough as possible. If there is something you cannot find in this manual, please check the knowledge base first. If you still cannot find an answer to your question, you may contact the tech support staff through the forums or use the priority support request form. We have tried to make the information in this manual as accurate as possible, however there may be some errors. If you find one, please let us know so we can correct it.

If you have just had AutoGallery SQL installed for the first time, please see the First Steps section of this manual first. It will help get you started using AutoGallery SQL and will guide you through the first things you need to do.
